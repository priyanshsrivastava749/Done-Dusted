How to get your YouTube API Key

To import playlists, this app needs a YouTube Data API v3 key from Google.

  1. Go to the Google Cloud Console.
  2. Create a New Project (e.g. named "Done-Dusted").
  3. In the search bar, type "YouTube Data API v3" and select it.
  4. Click Enable.
  5. On the left sidebar, click Credentials.
  6. Click + CREATE CREDENTIALS and select API key.
  7. Copy the generated key (starts with AIza...).
  8. Paste it in your Dashboard.

Note: This key is stored locally in your database and used only to fetch playlist details.
